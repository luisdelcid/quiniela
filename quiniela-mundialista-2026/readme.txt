Plugin instalable para gestionar quinielas privadas del Mundial FIFA 2026 dentro de WordPress.

== Instalación ==
1. Sube la carpeta quiniela-mundialista-2026 a wp-content/plugins/.
2. Activa el plugin en Plugins.
3. El plugin crea las tablas, carga datos base y crea la página "Quiniela Mundialista".

== Uso rápido ==
* Crea una quiniela en Quiniela Mundialista > Quinielas.
* Comparte el código privado con participantes.
* Usa [quiniela_mundialista] para landing y acceso.
* Usa [quiniela_mundialista_predicciones] para capturar marcadores.
* Usa [quiniela_mundialista_ranking] para ranking.
* Usa [quiniela_mundialista_mis_puntos] para detalle individual.
* Usa [quiniela_mundialista_bracket] para bracket simple.

== Resultados y puntajes ==
Carga resultados en Quiniela Mundialista > Partidos, marca el partido como finalizado y guarda. El plugin recalcula el partido. También existe el botón para recalcular todo.

== Importar / Exportar ==
Importa fixtures JSON desde la pantalla Importar/Exportar. Exporta ranking CSV por quiniela.
